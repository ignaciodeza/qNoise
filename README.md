# qNoise

Non-gaussian non white random noise generator. Useful as an input for numerical simulations, as a source of noise for controlling experiments and for a wide variety of applications.

# Usage

It is a stand alone library with no dependencies other than the standard libraries
Tests and examples are provided along with a python tool to generate most of the figures of the article.

Just install it near to your code and include.
